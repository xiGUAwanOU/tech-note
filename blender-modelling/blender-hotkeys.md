# Blender Hotkeys

All the hotkeys are default in MacOS systems, may be a little bit different in other systems.

# General
* `TAB`: toggle edit mode;

# Object Select
* `A`: toggle select all;
* `B`: different select modes;
* `L`: select whole mesh;

# Object Transform
* `G`: grab;
* `R`: rotate;
* `S`: scale;

# Object Edit
* `E`: extrude;
* `F`: fill line/face;
* `X`: delete;
