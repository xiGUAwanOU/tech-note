# Working with Tags

To create a tag:

```console
$ git tag -a 0.0.1 -m "Release the very first version of the project."
```

To push a tag:

```console
$ git push origin 0.0.1
```

To checkout a tagged version:

```console
$ git checkout tags/<TAG_NAME>
```

To go back to master branch:

```console
$ git checkout master
```
