# Modifier Order

The modifiers in Java source codes should follow this order according to [JLS](http://docs.oracle.com/javase/specs/jls/se8/html/jls-8.html):

  1. public
  2. protected
  3. private
  4. abstract
  5. static
  6. final
  7. transient
  8. volatile
  9. synchronized
  10. native
  11. strictfp
