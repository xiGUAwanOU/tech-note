About Column

Here, you may wonder, why there are so many xs, sm, ... in the code.

They are actually used for responsive design:

Level Applied Width
xs <768px;
sm ≥768px;
md ≥992px;
lg ≥1200px;

 

More About Column

What is about the number after col-sm-, for example col-sm-6?

There are 12 column stops in 1 row in Bootstrap. So basically the number after col-sm- is how many column stops this column will take. Usually, the numbers of columns in one row should have a sum of 12. Otherwise, either they won't take the full width of the container, or one or more column will displayed in next line.

For example, if we want 3 columns with the same width, we could make all the 3 column col-sm-4, so that 3 * 4 = 12.

Make It Responsive

We should order the styles from smaller screen to larger screens. For example, if I want these 3 columns in the row stacked on screens narrower than 768px, I should write following code: 

<div class="col-xs-12 col-sm-4">
Note that, there is a col-xs-12, which means if the screen is narrower than 768px, this column will take the full row by taking 12 column stops; otherwise col-sm-4 makes it take 1/3 of a row.
